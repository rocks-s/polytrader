Телеграм бот, позволяющий включать автотрейдинг на Polymarket с с различными опциями. Бот имеет подписочную систему, в которой платежи принимаются криптовалютой

ENG
A Telegram bot that allows you to enable automated trading on Polymarket with various options. The bot has a subscription system that accepts payments in cryptocurrency.
Before launching, look at /config, then run python main.py in the root directory.
